- Abstract

심리학자들이 자신의 연구를 진행할 때 자극을 제공하고 반응을 관찰하는 형식의 실험 설계가 필요합니다.
그들은 이러한 실험을 만들기 위해 프로그래밍을 해야 했으며, 그것을 배포하고, 또 실험의 결과를 이메일로 받아서
취합하는 등의 번거로움이 있었습니다.

이 프로젝트는 임의의 심리학자들이 자신들이 원하는 심리학 실험을 손 쉽게 생성하고, 배포할 수 있게 해주는
웹 서비스 입니다. 당신이 원하는 실험의 명세를 txt 파일로 만들어서 업로드하기만 하면, 우리가 당신이 의도한
실험을 생성하고 도메인을 부여합니다. 따라서 당신은 번거롭게 게임을 생성할 필요가 없을 뿐만 아니라, 그것을
배포하기 위해 프로그램을 전달할 필요도 없습니다. 생성된 실험의 도메인을 피실험자들에게 메신저로 전송하기만
하면 됩니다! 그들의 실험은 웹 브라우저에서 수행되며 실험의 결과가 우리 서버의 데이터베이스에 저장됩니다.
당신은 당신이 원하는 아무때나 실험의 결과를 조회하고 다운로드할 수 있습니다.

그들이 우리의 프레임워크를 이용해 생성할 수 있는 실험들의 예시가 제공됩니다.

또한, 실험중에 피실험자의 감정을 웹 캠이나 마이크로 인식할 수 있는 딥러닝 기능도 제공합니다.


- Background

이 프로젝트의 기술 스택은 다음과 같습니다.

Frontend - HTML / CSS / Javascript / Bootstrap / jQuery
Backend - Python / Django / MySQL / Apache2 / AWS Linux Instance

Frontend 는 편명장이, Backend 는 김유민이 개발했습니다.


- Specifications

이 프로젝트는 3 개의 장고 앱으로 구성되어 있습니다.
1. game 앱
 게임 앱은 당신이 이 프레임워크를 사용하여 생성할 수 있는 많은 게임들을 예시로써 보여줍니다.

2. researcher 앱
 실험자 앱에서 당신은 당신만의 프로젝트(여러 게임들을 시퀀스로 만든 것)를 만들고 배포할 수 있습니다.
또한 저희 데이터베이스에 저장된 실험의 결과를 쉽게 확인하고 다운로드할 수 있습니다.

3. ML 앱
 ML 은 머신러닝의 약자로, 이 프로젝트에서 인공지능을 사용하는 부분을 맡습니다.
